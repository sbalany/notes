How to get repo synced onto a new computer:
- Login to github.com using your personal account (sbalany).
- Go to Settings -> Developer settings -> Tokens
- Copy the API token for the Obsidian App.
- In Obsidian, go to Community Plugins -> Search for Git plugin by Vinzent. Install it. Enable it.
- Ctrl + P -> clone. In remote URL, type "https://TOKEN@github.com/sbalany/notes.git". Local folder: "notes". Full-depth.
- Done!!
